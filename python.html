Basics Of Python...

Let Us Learn Some Basics First
  • Defination:
     Python is an interpreted high-level general-purpose programming language. Its design philosophy emphasizes code readability with its use of significant indentation. Its language constructs as well as its object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.
    Installation:

      1. Download python executable file by clicking HERE or visiting their official website python.org
      2. Open the downloaded exe file

      3. Check Add Python 3.9 to PATH box.

      4. Wait Until Successful Installation.


    Running:

      1. Search for IDLE (python 3.8 64bit) software
      2. Open IDLE

      3. Click on File --> New File or directly press ctrl+N

      4. A new window will open and write any python code you need

      5. Save the written python code by clicking file --> save or directly press ctrl+S

      6. Run your code by clicking Run --> Run Module or directly pressing F5

      7. A new window will appear. Give the required input and check the output.

Arithmetic Operations Like Addition Subtraction Multiplication and Division
  • Addition: 
    x = input("Type a number: ")
y = input("Type another number: ")

    Subtraction:     
    x = input("Type a number: ")
