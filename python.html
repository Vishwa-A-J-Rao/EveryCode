Palindrome

To Check The Given Number Is Palindrome Or Not 
  • def flip(number):
    reverse = 0
    while(number > 0):
        Reminder = number % 10
        reverse = (reverse * 10) + Reminder
        number = number // 10
    return reverse
num = int(input("Please Enter any Num: "))
rev = flip(num)
if(num == rev):
    print(num,"is a Palindrome")
else:
    print(num,"is NOT a Palindrome")

Bubble Sort

Sort The Numbers In Accessing Order 
  • a = []
num = int(input("Please Enter the Total Number of Elements : "))
for i in range(num):
    value = int(input("Please enter the %d Element of List1 : " %i))
    a.append(value)
for i in range(num -1):
    for j in range(num - i - 1):
        if(a[j] > a[j + 1]):
             temp = a[j]
             a[j] = a[j + 1]
             a[j + 1] = temp
print("The Sorted List in Ascending Order : ", a)
